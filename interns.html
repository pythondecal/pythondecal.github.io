Interns

We offer paid internship positions for the Python DeCal. Students that are paid interns shadow the current staff and learn how to do all the tasks required to run the course. This entails grading, writing problem sets, creating presentations, giving lectures, and hosting office hours. Please send us an email if you are interested in an internship for the Python DeCal.
We currently do not have interns since we changed the course and would like to nail it before adding additional members to our team.
